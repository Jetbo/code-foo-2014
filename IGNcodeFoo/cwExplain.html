Crossword Explanation


Basic Breakdown:

  1. Test user input for errors
  2. Sort words by length
  3. Convert input into all uppsercase
  4. Place first word onto a 2D grid
  5. Iterate through words
    1. Take each word and iterate through its individual letters
    2. Compare that letter to every letter on the grid
    3. If a letter matches, test to see if the whole word will fit on the grid
      • FITS: Place word on grid
      • DOES NOT FIT: Do not place word on grid and put into a "failed" que
    4. Remove word from que and repeat step #5 till words run out
    5. Take the "failed" que and repeat step #5 till words run out
      • Failed words go into a new "final failed" group
  6. Convert 2D grid into table system
  7. Output table
  8. Output list of final failed words
